用法

Clone（下载）项目，在Calcore项目上右键生成，会生成. dll文件。
在你的项目内引用这个dll文件.

方法

string formula;
string result="";

lowCal只限于加减法运算 result=lowCal(formula);
Multiply加减乘除，乘方 result=Multiply(formula);

Demo

Using Cal_core;

CalCore Cal = new Cal;
double result="";

加减
result=lowCal("1+1.1");

乘方:
result=Multiply(3^3);
