昨天重新灌了电脑的系统，发现 hexo 不能用了

安装基本软件

  • Git
  • Node.js

注意：必须使用 Node v12.14.0 旧版 Hexo 才不会报错

注:Node.js(中文网站)没有使用 https，原因是使用后可能导致无法打开此网页

步骤

假使 hexo 目录为 D:/hexo

  • 打开 CMD [按下**(win+r)**组合键，输入 cmd]
  • 输入 npm:如果没有 npm 的详细信息，请重装 Node.js
  • CMD 中定位到 hexo 目录
1d:
2cd hexo
  • 安装 hexo:在 CMD 中输入npm install -g hexo
  • 安装完成后关闭 CMD
  • 打开 Git，定位到 hexo 的目录cd d:/hexo
  • 可以使用啦!

密钥问题

打开 Github，转到你的设置 找到SSH and GPG keys，添加新的密钥

1git config --global user.name "用户名"
2git config --global user.email "邮箱"
3
4ssh-keygen -t rsa -C "邮箱"

你的 windows 账户文件夹内，会有.ssh 的文件夹，找到里面的id_rsa.pub文件。将其内部所有内容复制到 github 的密钥框中，确定即可。

注:如果上面配置 ssh 时叫你设置密码设置空密码就可以了。
